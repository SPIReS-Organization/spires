SPIReS — SPectral Inversion of REflectance from Snow
=====================================================

This is the front door to the `SPIReS <https://github.com/SPIReS-Organization>`_
package family. It is an **umbrella site**: it explains what SPIReS is, how the
packages fit together, and how to install them — then links out to each
package's own documentation for API detail.
